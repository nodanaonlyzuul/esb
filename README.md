# Esb

Look up the Empire State Building's current color - from your command line.

## Installation

    $ gem install esb

## Usage

    $ esb

    November 25 - 28
    Special Lighting in Fall Colors
    In honor of the 87th Annual Macy's Thanksgiving Day Parade
